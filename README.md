# MontgoPulse 🚀
### Strategic Urban Intelligence & Investment Analytics Platform

MontgoPulse is a strategic urban intelligence platform designed to bridge the gap between raw city data and high-stakes investment decisions. By synthesizing real-time spatial analytics with predictive market modeling, it empowers developers and stakeholders to pinpoint undervalued zones, simulate capital impact, and uncover prime development opportunities.

## 🌟 Key Features
* **Live Geospatial Mapping:** Integration with ArcGIS for real-time tracking of construction permits, business licenses, and public safety trends.
* **AI Strategic Advisor:** A built-in expert agent (powered by Gemini) that analyzes specific parcels, predicts market gaps, and provides investment narratives.
* **Investment Simulator:** Scenario-based modeling to calculate the viability of residential, commercial, or industrial developments.
* **Automated Briefings:** Instant generation of professional, data-driven reports for stakeholders and institutional investors.

## 🛠️ Tech Stack
* **Framework:** Next.js 14 (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Data Pipelines:** ArcGIS REST APIs & Bright Data
* **AI Orchestration:** Google Gemini 1.5 Flash
* **Deployment:** Vercel

## 🚀 Getting Started

### Prerequisites
* Node.js 18.x or later
* ArcGIS Developer Account
* Gemini API Key

### Installation

**1. Clone the repository**

```bash
git clone https://github.com/your-username/montgopulse.git
cd montgopulse
```

**2. Install dependencies**

```bash
npm install
```

**3. Set up environment variables**

```bash
cp .env.example .env.local
```

Then open `.env.local` and fill in your API keys and ArcGIS endpoints (see [Environment Variables](#-environment-variables) below).
